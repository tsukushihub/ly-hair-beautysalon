## Netlifyって、結局何なの？

一言で言うと、**作ったHPをインターネット上に公開するためのサービス**です。

「サーバー」と呼ばれるインターネット上の場所にファイルを置くことで、世界中からHPにアクセスできるようになります。普通はサーバーを借りるのにお金がかかりますが、Netlifyなら**無料**で使えます。

**Netlifyのここがすごい：**
- 無料でHTTPS（鍵マーク）に対応している
- **商用利用OK**（お店や仕事のHPに使える）
- GitHubと連携すれば、ファイルを更新するだけで自動でHPに反映される
- 独自ドメイン（自分だけのURL）も設定できる

> 💡 「GitHub Pages」という似たサービスもありますが、商用利用が禁止されています。お店や仕事のHPにはNetlifyを使いましょう。

## STEP 1：Netlifyのアカウントを作る

まずは netlify.com にアクセスします。

<!-- 📸 Netlifyトップページのスクリーンショットをここに挿入 -->

1. 「**Sign up**」をクリック
2. 「**Sign up with GitHub**」を選ぶと後の連携がスムーズです
3. GitHubのアカウントでログインを許可する
4. アカウント作成完了！

<!-- 📸 Netlifyサインアップ画面のスクリーンショットをここに挿入 -->
<!-- 📸 GitHubでのログイン許可画面をここに挿入 -->

> 💡 GitHubアカウントでサインアップしておくと、後でGitHubとの連携がワンクリックでできます。

## STEP 2：GitHubと連携してサイトを公開する

アカウントができたら、さっそくHPを公開しましょう。

<!-- 📸 Netlifyダッシュボード画面をここに挿入 -->

1. ダッシュボードの「**Add new site**」をクリック
2. 「**Import an existing project**」を選択
3. 「**GitHub**」をクリック

<!-- 📸 GitHubとの連携認証画面をここに挿入 -->

4. 連携の許可を求める画面が出るので「**Authorize Netlify**」をクリック
5. リポジトリ一覧が表示されるので、公開したいリポジトリを選択

<!-- 📸 リポジトリ選択画面をここに挿入 -->

6. デプロイの設定画面が表示される（そのままでOK）
7. 「**Deploy ○○ to Netlify**」をクリック

<!-- 📸 デプロイ設定画面をここに挿入 -->

数十秒後、サイトが公開されます！

<!-- 📸 デプロイ完了画面をここに挿入 -->

表示されたURLをクリックして、自分のHPが開けば成功です。

> ✅ **ここがポイント**：一度連携すれば、次からGitHubにpushするだけで自動でサイトが更新されます。毎回Netlifyの操作は不要です。

## STEP 3：URLをお店の名前に変える

Netlifyが自動でつけるURLは `cheerful-starburst-〇〇〇〇.netlify.app` のようなランダムな名前です。これをわかりやすい名前に変えられます。

<!-- 📸 変更前のURL画面をここに挿入 -->

1. Netlifyのダッシュボードでサイトを開く
2. 左サイドバーの「**Project configuration**」をクリック
3. 「**General project settings**」を下にスクロール
4. 「**Project name**」の横にある「**Change project name**」をクリック
5. 好きな名前（例：`my-shop`）を入力して保存

<!-- 📸 URL変更後の画面をここに挿入 -->

他の誰かが使っていなければ、`https://my-shop.netlify.app` というスッキリしたURLになります。無料で今すぐできます！

## STEP 4：HPの内容を更新したいときは？

営業時間や料金など、HPの内容を後から変更したくなることは必ずあります。GitHubと連携していれば、**3ステップで更新完了**です。

**① ファイルを編集して保存する**

`index.html` をテキストエディタ（メモ帳・VSCodeなど）で開き、変更したい部分を書き換えて保存します。

**② PowerShellで以下を実行する**

```powershell
# プロジェクトのフォルダに移動
cd フォルダのパス

# 変更ファイルをステージング
git add index.html

# 変更内容を記録
git commit -m "営業時間を更新"

# GitHubに送信
git push
```

**③ 数分待つと自動で反映される**

pushが完了すれば、Netlifyが自動で検知して数分以内にHPへ反映されます。ブラウザで確認してみましょう。

> 💡 **ポイント**：Netlifyの管理画面をまったく触らなくていい、というのが最大のメリットです。
