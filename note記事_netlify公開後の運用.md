## STEP 1：詰まりポイント① 画像が出ない！Windowsの罠にハマりました

公開したのにサイトを開いたら画像がすべて表示されない…。

**原因：Windowsの「拡張子の二重表示」問題**

Windowsは初期設定で「既知のファイルの拡張子を表示しない」設定になっています。そのため：

<!-- 📸 画像⑫（ファイル一覧：exterior.jpg など）をここに挿入 -->

- 画面上では `exterior.jpg` に見えているファイルが…
- 実際のファイル名は `exterior.jpg.jpg` になっている！

HTMLコードでは `exterior.jpg` と書いているのに、実際のファイルは `exterior.jpg.jpg`。だから画像が読み込めなかったのです。

**解決方法：**

エクスプローラーを開いて「表示」→「ファイル名拡張子」にチェックを入れます。本当のファイル名を確認して、コードのファイル名と一致させれば解決します。

> 💡 Windowsユーザーは要注意！画像が表示されない場合はまずファイル名を疑いましょう。

## STEP 3：公開して終わりじゃない！集客に繋げる一工夫を

サイトを公開したら、知ってもらう工夫も大切です。

### QRコードを作る

<!-- 📸 画像㉒（Ly hairのQRコード）をここに挿入 -->

名刺・店内POP・チラシに載せるために、URLをQRコード化するのが最も効果的です。無料の「QRコード作成サイト」にURLを入力するだけで、すぐに画像が完成します。レジ横に置いたり、ショップカードに印刷して配布するのがおすすめですよ。

### InstagramのプロフィールにURLを貼る
美容室のお客様はInstagramから来ることが非常に多いため、プロフィールの設定は欠かせません。編集画面からURLを貼るだけで作業は完了。ストーリーズで「新しいサイトができました！」と告知するのも効果的ですよ。

### Google検索に「見つけてもらう」ための工夫

サイトを公開した直後は、まだGoogleがあなたのサイトの存在を知りません。少しでも早く検索結果に表示されるように、以下の対応もAIと一緒に行いました。

- **サイトマップ(sitemap.xml)の作成**: Googleのロボットに「このサイトにはこんなページがありますよ」と教えるための地図です。
- **robots.txtの設置**: ロボットを正しくサイトへ案内するための指示書です。

これらを設置しておくことで、自然にGoogleに見つけてもらうための準備が完璧に整いました。

## 番外編：後からHPの内容を直したくなったら？

「営業時間が変わった」「料金を更新したい」など、公開後に内容を修正したくなることは必ずあります。GitHubと連携していれば、ファイルを編集してpushするだけで自動的に反映されます。

### 手順

**① ファイルを編集する**

`index.html` をテキストエディタ（メモ帳・VSCodeなど）で開き、変更したい部分を直接書き換えて保存します。

**② ターミナルで以下を順番に実行する**

```powershell
# プロジェクトフォルダに移動
cd プロジェクトのフォルダパス

# 変更したファイルをステージング
git add index.html

# 変更内容にメモをつけて保存
git commit -m "営業時間・料金を更新"

# GitHubに送信
git push
```

**③ 数分待つとNetlifyに自動反映される**

pushが完了すれば、Netlifyが自動で検知して数分以内にサイトへ反映されます。ブラウザでHPを開いて確認しましょう。

> 💡 **ポイント**：一度GitHubとNetlifyを連携しておけば、あとはpushするだけ。専門知識がなくてもHPの管理・更新ができます。
