# Ly hair & beautysalon ウェブサイト公開・運用ガイド

このドキュメントには、ウェブサイトの作成から公開、そして今後の運用方法までの手順をまとめています。

---

## 1. 準備と構築
- **構成**: HTML5 / CSS3 / JavaScript (Vanilla)
- **商用対応**:
    - 検索エンジン最適化（SEO）とSNS共有設定（OGP）の実装。
    - Googleマップや予約サイト（Hot Pepper Beauty）へのリンク。
    - プライバシーポリシーページの作成。

## 2. GitHubへのアップロード
ローカル（パソコン内）にあるコードを、世界中に公開するために GitHub に送信しました。

### 使用した主なコマンド（日常の更新も同じ）
```powershell
# 1. 変更したファイルを全てステージング
git add .

# 2. 変更内容にメモをつけて保存
git commit -m "修正内容のメモ"

# 3. GitHubに送信
git push origin main
```

## 3. Netlify での公開手順
GitHub にあるコードを Netlify に連携させることで、自動公開される仕組みを構築しました。

1.  **Netlify にログイン**: GitHub アカウントでログイン。
2.  **リポジトリの連携**: `ly-hair-beautysalon` を選択。
3.  **公開**: 自動でURL（`https://...netlify.app`）が発行されます。
4.  **URLの変更**: `Project configuration` > `Change site name` から、`ly-hair` という好きな名前に変更しました。

## 4. 運用上の重要な注意点（トラブル解決の教訓）

### 画像のファイル名について
Windowsの設定で「拡張子を表示しない」になっていると、`exterior.jpg` と名付けたつもりが実際には `exterior.jpg.jpg` になってしまうことがあります。
- **対策**: コード（HTML/CSS）側の記述と、実際のファイル名が1文字も違わず一致している必要があります。

### スマホでの表示調整
スマホで見ると文字が端に寄りすぎたり、変な場所で改行されることがあります。
- **対策**: CSSで `.container` に左右のパディング（余白）を設け、長い文章には `span` タグなどを使って言葉の区切りを保護しました。

## 5. 今後の更新方法
1.  パソコンでファイルを修正・保存する。
2.  上記の「GitHubへの送信コマンド」をターミナルで実行する。
3.  1〜2分待つと、自動的に `https://ly-hair.netlify.app` が更新されます。
